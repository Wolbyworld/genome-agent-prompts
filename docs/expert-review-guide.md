# Expert Review Guide

Use GitHub issues to review prompts by specialty.

## Clinical Genetics

Focus on ACMG/AMP usage, ClinVar interpretation, secondary findings, inheritance framing, carrier
status, VUS wording, and confirmation language.

## Pharmacogenomics

Focus on PharmCAT inputs, CPIC/DPWG wording, CYP2D6 structural variation, HLA typing limits, and
whether recommendations are framed as confirm-before-prescribing.

## PRS and Statistical Genetics

Focus on score selection, reference panels, ancestry limits, allele harmonization, palindromic SNPs,
coverage, standardization, uncertainty, and plain-language bands.

## Bioinformatics Reproducibility

Focus on build compatibility, gVCF handling, normalization, module versions, tool invocation,
temporary storage, and output provenance.

## Privacy and Safety

Focus on whether the prompt can accidentally disclose private data, overstate evidence, or encourage
remote processing without consent.
