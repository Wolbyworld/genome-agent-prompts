# Prompt Catalog

Each prompt is a reusable template. Replace placeholders with local values before running in a
private environment. Do not paste private genome data or generated result files into public issues.

- `clinical-acmg.md`: clinical variant safety pass with ACMG/ClinVar boundaries.
- `pharmacogenomics-cyp2d6-hla.md`: PGx profile with CYP2D6 and HLA gap-fill.
- `disease-prs.md`: standardized disease PRS workflow.
- `protective-traits.md`: protective and benign single-variant scan.
- `fun-polygenic-traits.md`: non-medical trait PRS workflow.
- `fitness-prs.md`: fitness and performance-related PRS workflow.
- `fine-ancestry.md`: fine ancestry and admixture workflow.
- `skeptical-wellness-report.md`: wellness genetics report with evidence grading.
